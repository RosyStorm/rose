## 你好，欢迎来到我的博客

这里记录我关于**技术**、**生活**与**思考**的随笔。

### 写些什么

- **技术笔记**：踩过的坑、用过的工具、读源码的笔记
- **生活随笔**：走过的地方、读过的书、看过的演出
- **方法论**：关于时间管理、知识管理、效率提升的一些实践

### 关于评论

每篇文章底部都启用了 [Giscus](https://giscus.app/zh-CN) 评论系统，基于 GitHub Discussions，无需注册即可用 GitHub 账号留言。

### 关于部署

本站使用 [Jekyll](https://jekyllrb.com/) + [Academic Pages](https://github.com/academicpages/academicpages.github.io) 主题，部署在 [GitHub Pages](https://pages.github.com/) 上。所有文章以 Markdown 形式存放在 `_posts/` 目录，提交到 `master` 分支后 5 分钟内自动构建并发布。

### 联系方式

欢迎通过以下方式与我交流：

- 在任意文章下方留言
- 邮件：your@email.com
- GitHub：[@yourusername](https://github.com/yourusername)
